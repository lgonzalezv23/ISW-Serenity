Insomnio

El insomnio es un trastorno frecuente del sueño. En el insomnio, tiene dificultades para conciliar el sueño, para mantenerse dormido o para lograr un sueño de buena calidad. Esto ocurre incluso cuando tiene el tiempo y el entorno correcto para dormir bien. El insomnio puede interferir con sus actividades diarias y puede hacer que se sienta somnolencia durante el día.

El insomnio puede afectar la memoria y la concentración. El insomnio crónico aumenta el riesgo de tener presión arterial alta, cardiopatía isquémica, diabetes y cáncer.
